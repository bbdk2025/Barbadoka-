# 巴巴多卡工具箱

一个功能丰富的实用工具集合，支持在电脑和手机上使用，提供多种日常生活和工作所需的工具。

## 📋 功能列表

### 🔍 信息查询工具

- **邮编代码查询** - 选择地区查询对应邮编（支持区县级别）
- **当前油价查询** - 获取全国各省份实时油价数据
- **本机IP查询** - 自动检测并显示公网IP地址

### 🖼️ 图片处理工具

- **图片格式转换器** - 支持转换为PNG、JPG、WebP格式
- **图片压缩器** - 调整压缩质量并下载
- **九宫格切图** - 将图片分割为九宫格并支持下载
- **表情包制作** - 添加文字生成个性化表情包

### 🎥 视频工具

- **视频转GIF** - 将视频转换为GIF动画

### 🔧 工具生成器

- **生成二维码** - 根据文本或URL生成二维码
- **生成摩斯密码** - 将文本转换为摩斯密码

### 💱 金融工具

- **汇率计算器** - 支持多种货币之间的汇率转换
- **BMI计算器** - 计算身体健康指数并显示状态

### 👨‍👩‍👧‍👦 生活工具

- **亲戚关系计算器** - 计算不同角色之间的亲戚关系

## 🚀 快速开始

### 方法一：使用链接
- 访问链接：<https://bbdk2025.github.io/Barbadoka-tools/>即可使用

### 方法二：下载运行

1. **下载项目**
   - 访问 GitHub 仓库：<https://github.com/bbdk2025/Barbadoka-tools>
   - 点击 "Code" 按钮，选择 "Download ZIP" 下载压缩包

2. **解压文件**
   - 将下载的ZIP文件解压到任意目录

3. **直接运行**
   - 双击解压目录中的 `index.html` 文件
   - 系统会自动用默认浏览器打开
   - 即可开始使用工具箱

4. **创建桌面快捷方式**
   - 右键点击 `index.html` 文件
   - 选择 "发送到" > "桌面快捷方式"
   - 桌面会生成一个快捷方式，双击即可打开


## 🔄 自动更新功能

- 应用启动时会自动检查GitHub上的最新版本
- 如果发现新版本，会弹出更新提示
- 点击"更新"按钮会跳转到GitHub下载页面
- 下载最新版本后，解压并替换旧文件即可完成更新
